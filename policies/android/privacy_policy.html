  • Device information (model, operating system version)

1.2 Chrome Extension (Rehan Job Helper)

Privacy-First Design: The Rehan Job Helper extension does NOT collect, transmit, or share any personal data with external servers or third parties.

Local Storage Only: All data is stored locally in your browser using Chrome's storage API:

  • Form field data you choose to save
  • Job application tracking information
  • Extension settings and preferences

Chrome Sync Storage: If you enable Chrome Sync in your browser, your saved data will sync across your devices through your Google account. This is handled entirely by Chrome and Google - we do not have access to this data.

5. Chrome Extension Permissions

The Rehan Job Helper extension requests the following permissions:

5.1 Storage Permission

Purpose: To store user-saved form data and job application details locally in the browser and sync across devices (if Chrome Sync is enabled).

Data Scope: Only data you explicitly save is stored.

5.2 Active Tab Permission

Purpose: To interact with the current webpage only when you explicitly click "Save" or "Auto Fill" buttons.

Data Scope: Access is limited to the active tab and only when triggered by user action.

5.3 Side Panel Permission

Purpose: To display the extension's management interface for viewing and organizing saved fields and job applications.

Data Scope: Interface only, no additional data access.

5.4 Host Permissions (all_urls)

Purpose: To work on various job portals and application websites where users manually save and fill form data.

Data Scope: The extension only runs when you enable it and interact with it. It does not operate automatically in the background or collect data without your explicit action.
